# always

- **End every run with a conventional commit.** A run that decided nothing still commits —
  the decision to change nothing is the output.
- **On partial failure, write a stub.** If a source died, a tier degraded, or an input file
  was missing, write a file naming the failure and commit it. A gap must be visible in the
  repo. Silence is the one unacceptable outcome.
- **Cite or drop it.** Every claim about demand names the candidate it came from, by URL.
- **Say when nothing happened.** "No candidate out-argued an incumbent tonight" is a
  complete, valid, useful run.
- **Write dates absolute.** `2026-07-16`, never "yesterday".
